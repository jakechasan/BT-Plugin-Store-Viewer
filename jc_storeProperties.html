Finding the iTunes Item ID:
1. Obtain the URL for the iTunes Item (ex: https://itunes.apple.com/us/app/trivia-game-for-downton-abbey/id624628271?mt=8)
2. The Item Id is within the URL. You will find the number after the "id" in the URL (ex: 624628271)
3. Place the number into the "iTunes Item ID" box in the Control Panel, be sure the only thing you place within the box is numbers.
4. Click "Save"
